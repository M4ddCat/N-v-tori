#Setup

from cmd:

````
git clone https://github.com/M4ddCat/N-v-tori %USERPROFILE%\AppData\Local\nvim && nvim
````

from powershell:

````
git clone https://github.com/M4ddCat/N-v-tori $ENV:USERPROFILE\AppData\Local\nvim; nvim
````
